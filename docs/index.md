# Welcome to Motion Docs

Motion is a powerful timeline animation plugin for Roblox.

## Quickstart

1. Install Motion from Roblox Marketplace  
2. Create a new project and add tracks  
3. Add keyframes and preview your animation  
4. Export and use the runtime engine to play animations in-game
